Search for local land charges on land and property

Local land charges restrict the way you use land and property.
Examples include:

  • planning decisions
  • road agreements
  • tree preservation orders
  • conservation areas
  • listed buildings
  • environmental health notices

Use this service to check for local land charges in England and Wales.

This service is also available in Welsh (Cymraeg).

Not all data is on this service yet. You can check if a local authority has added their data first.

Before you start

You'll need to create an account to use this service.

To search for a local land charge, you will need to identify a property or area of land. We call this a search area.

You can search by postcode and street name, coordinates, map, title number and INSPIRE ID.

It's free to do a search and you can pay £15 for an official search certificate.

You can contact HM Land Registry (opens in new tab) with any queries about using this service.
