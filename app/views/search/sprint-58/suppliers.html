HM Land Registry: Local land charges software suppliers

[The following software suppliers are contracted to work with HM Land Registry to support local authorities with the Local land charges service.]

From: HM Land Registry
Published 31 August 2017

Applies to England and Wales

The following software suppliers are contracted to work with HM Land Registry to support local authorities with the Local land charges service

HM Land Registry does not recommend or endorse any one product or service over another and will not be responsible for any loss, damage, cost or expense in connection with using this software.

Please contact the software supplier to:

  • ensure the software product meets your specific needs
  • provide support
  • request further information about the software
