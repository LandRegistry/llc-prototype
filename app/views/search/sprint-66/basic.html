Help using the map with a keyboard

You can select an area from the map and delete areas using a keyboard only. The keyboard focus needs to be on the map to do this.

You can not draw or edit an area using a keyboard.

You can pan using the arrow keys. Fine-tune the position by selecting the alt key in Windows, or the option key on a Mac.

Select the area at the centre of the map using the space bar.

Focus the map using access key M (ctrl + option/alt).
