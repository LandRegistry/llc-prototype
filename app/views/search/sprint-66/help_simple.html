How to use the map

How to select an area using copy from map

You can only select an area from the map when you are zoomed in to a location (seeing a place close-up). This option will be inactive until you are zoomed in enough. Make sure you can see building outlines on the map.

In the video, a user:

  • Selects the select area from map radio button.
  • Selects different map areas one after the other to create a combined search area.

Before confirming, ensure your search area includes all of the properties and land you want to search against, including any driveways, alleys or gardens, etc. Using the select area from map feature once may not capture all of your search area.

How to draw your search area

In the video, to draw a search area with straight edges, a user:

  • Selects the draw area radio button.
  • Selects a place on the map to start drawing.
  • Keeps selecting around the edges of the search area.
  • Joins the points to stop drawing.

In the video, to draw an extent with curved edges, a user:

  • Selects the draw area radio button.
  • Selects a place on the map to start drawing.
  • Hold the SHIFT key and drag the cursor to draw a curved search area.
  • Release the SHIFT key and mouse button to stop drawing.

Before confirming, ensure the search area captured includes all of the properties and land you want to search against, including (if necessary) driveways, alleys or gardens, etc.

How to edit your search area

In the video, a user:

  • Selects the edit area radio button.
  • Sees the search area turn green.
  • Selects one of the points.
  • Drags the point to change the search area.
  • Holds ALT key while selecting another point to delete it.

How to draw a shape with snap to

You can only use snap to line on map when you are zoomed into a location (seeing a place close-up). This option will be inactive until you are zoomed in enough.

In the video, a user:

  • Selects the draw area radio button.
  • Selects the snap to line on map checkbox.
  • Hovers the cursor near where they want to begin drawing their search area.
  • Sees the point snapping to a line on the map as they hover close to it.
  • Selects to add multiple points to draw their search area.
  • Joins the points to stop drawing.

This feature can also be used during other map interactions, like editing.

How to delete a search area

In the video, a user:

  • Selects the delete area radio button.
  • Sees the search area turn red.
  • Selects two parts of the search area to remove them.

How to clear all

In the video, a user:

  • Selects the clear all button.
  • Sees everything they have drawn on the map being deleted.

How to undo your last map interaction

In the video, a user:

  • Draws an area on the map.
  • Deletes an area.
  • Selects the undo button to undo deleting an area.
  • Selects the undo button again to undo drawing an area.

How to use the map with a keyboard only

Select an area from the map and delete areas using a keyboard only. The keyboard focus needs to be on the map to do this.

You can not draw or edit an area using a keyboard.

Keyboard controls

  • Arrow keys to move the map up, down, left and right.
  • SHIFT key with arrows to fine-tune the position.
  • Plus sign (+) to zoom in and Minus sign (-) to zoom out.
  • SPACE bar or ENTER key to select the area at the centre of the map.
